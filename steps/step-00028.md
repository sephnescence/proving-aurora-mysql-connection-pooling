# Step ✅ 00028 — Install @pooling-poc/integration-tests dependencies

## Commands

Run from the repo root:

```bash
pnpm --filter @pooling-poc/integration-tests add -D vitest typescript @types/node
```

## Note on version drift

Because no versions are pinned in the install command, pnpm will resolve the latest available at the time you run it. If this step is run significantly later than earlier steps, `typescript`, `vitest`, and `@types/node` may resolve to newer versions than those installed in other packages — this is expected and not intentional.

To check for version inconsistencies across the whole monorepo:

```bash
pnpm outdated -r
```

This shows all outdated packages across every workspace package. If you want to bring other packages up to the same versions, update them individually:

```bash
pnpm --filter <package-name> update typescript vitest @types/node
```

Or to update all packages across the entire monorepo at once (use with care):

```bash
pnpm update -r
```

## Expected result

`packages/integration-tests/package.json` has a `devDependencies` section with `vitest`.
