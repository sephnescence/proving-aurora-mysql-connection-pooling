# Step 00028 — Scaffold @pooling-poc/integration-tests package

## Context

Create the `packages/integration-tests` directory and initialise its `package.json`.

## Commands

```bash
mkdir -p packages/integration-tests/src
cd packages/integration-tests
pnpm init
```

Then edit `packages/integration-tests/package.json` so it contains exactly:

```json
{
  "name": "@pooling-poc/integration-tests",
  "version": "1.0.0",
  "private": true,
  "scripts": {
    "test": "vitest run"
  }
}
```

## Expected result

`packages/integration-tests/package.json` exists with the name `@pooling-poc/integration-tests`.
