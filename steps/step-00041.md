# Step 00041 — Create k8s/namespace.yaml

## Context
All Kubernetes resources for this PoC live in the `pooling-poc` namespace.

## Action
Create the file `k8s/namespace.yaml` with the following exact content:

```yaml
apiVersion: v1
kind: Namespace
metadata:
  name: pooling-poc
```

## Expected result
`k8s/namespace.yaml` exists.
