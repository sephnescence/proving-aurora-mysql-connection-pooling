# Step 00055 — Create .github/workflows/artillery-sequential.yml

## Context

Runs the Artillery sequential load test against the deployed ALB, then polls CloudWatch Logs Insights for 5 minutes and prints per-app connection count summaries.

## Action

Create the file `.github/workflows/artillery-sequential.yml` with the following exact content:

```yaml
name: Artillery — Sequential

on:
  workflow_dispatch:
    branches:
      - main

permissions:
  id-token: write
  contents: read

jobs:
  artillery-sequential:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - uses: pnpm/action-setup@v4
        with:
          version: latest

      - uses: actions/setup-node@v4
        with:
          node-version: '22'
          cache: 'pnpm'

      - name: Configure AWS credentials
        uses: aws-actions/configure-aws-credentials@v4
        with:
          role-to-assume: arn:aws:iam::${{ secrets.AWS_ACCOUNT_ID }}:role/aurora-pooling-poc-artillery-role
          aws-region: ap-southeast-2

      - name: Get stack outputs
        run: |
          STACK=aurora-pooling-poc
          get_output() {
            aws cloudformation describe-stacks --stack-name $STACK \
              --query "Stacks[0].Outputs[?OutputKey=='$1'].OutputValue" --output text
          }
          ALB_URL=$(kubectl get ingress pooling-poc-ingress -n pooling-poc \
            -o jsonpath='{.status.loadBalancer.ingress[0].hostname}' 2>/dev/null || \
            aws cloudformation describe-stacks --stack-name $STACK \
              --query "Stacks[0].Outputs[?OutputKey=='AlbUrl'].OutputValue" --output text)
          echo "ALB_URL=http://$ALB_URL" >> $GITHUB_ENV
          echo "POOLED_LOG_GROUP=$(get_output PooledLogGroupName)" >> $GITHUB_ENV
          echo "UNPOOLED_LOG_GROUP=$(get_output UnpooledLogGroupName)" >> $GITHUB_ENV

      - name: Install load-tests dependencies
        run: pnpm --filter @pooling-poc/load-tests install --frozen-lockfile

      - name: Record test start time
        run: echo "TEST_START_TIME=$(date +%s)000" >> $GITHUB_ENV

      - name: Run Artillery sequential
        working-directory: packages/load-tests
        run: pnpm exec artillery run sequential.yml
        env:
          TARGET_URL: ${{ env.ALB_URL }}/pooled

      - name: Poll CloudWatch Logs Insights
        working-directory: packages/load-tests
        run: |
          pnpm exec tsx poll.ts \
            --testType sequential \
            --startTime $TEST_START_TIME \
            --pooledLogGroup $POOLED_LOG_GROUP \
            --unpooledLogGroup $UNPOOLED_LOG_GROUP
```

## Expected result

`.github/workflows/artillery-sequential.yml` exists.
