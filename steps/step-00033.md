# Step 00033 — Scaffold @pooling-poc/load-tests package

## Context
Create the `packages/load-tests` directory and initialise its `package.json`.

## Commands

```bash
mkdir -p packages/load-tests
cd packages/load-tests
pnpm init
```

Then edit `packages/load-tests/package.json` so it contains exactly:

```json
{
  "name": "@pooling-poc/load-tests",
  "version": "1.0.0",
  "private": true,
  "scripts": {
    "artillery:sequential": "artillery run sequential.yml",
    "artillery:concurrent": "artillery run concurrent.yml"
  }
}
```

## Expected result
`packages/load-tests/package.json` exists with the name `@pooling-poc/load-tests`.
