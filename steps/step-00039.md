# Step 00039 — Create docker-compose.yml

## Context

Local development setup. Three services: `mysql` (MySQL 8.0), `pooled-app` (host port 3000), `unpooled-app` (host port 3001). Both NestJS apps depend on `mysql` being healthy before starting.

## Action

Create the file `docker-compose.yml` at the repo root with the following exact content:

```yaml
services:
  mysql:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: ${MYSQL_ROOT_PASSWORD}
      MYSQL_DATABASE: pooling_poc
      MYSQL_USER: admin
      MYSQL_PASSWORD: ${DB_PASSWORD}
    ports:
      - '3306:3306'
    healthcheck:
      test: ['CMD', 'mysqladmin', 'ping', '-h', 'localhost']
      interval: 10s
      timeout: 5s
      retries: 5
      start_period: 30s

  pooled-app:
    build:
      context: .
      dockerfile: apps/pooled-app/Dockerfile
    ports:
      - '3000:3000'
    environment:
      DB_HOST: mysql
      DB_PORT: '3306'
      DB_USER: admin
      DB_PASSWORD: ${DB_PASSWORD}
      DB_NAME: pooling_poc
      DB_SSL: 'false'
      DB_CONNECTION_MODE: persistent
    depends_on:
      mysql:
        condition: service_healthy
    healthcheck:
      test:
        [
          'CMD',
          'wget',
          '--no-verbose',
          '--tries=1',
          '--spider',
          'http://localhost:3000/health',
        ]
      interval: 10s
      timeout: 5s
      retries: 5
      start_period: 30s

  unpooled-app:
    build:
      context: .
      dockerfile: apps/unpooled-app/Dockerfile
    ports:
      - '3001:3000'
    environment:
      DB_HOST: mysql
      DB_PORT: '3306'
      DB_USER: admin
      DB_PASSWORD: ${DB_PASSWORD}
      DB_NAME: pooling_poc
      DB_SSL: 'false'
      DB_CONNECTION_MODE: per-request
    depends_on:
      mysql:
        condition: service_healthy
    healthcheck:
      test:
        [
          'CMD',
          'wget',
          '--no-verbose',
          '--tries=1',
          '--spider',
          'http://localhost:3000/health',
        ]
      interval: 10s
      timeout: 5s
      retries: 5
      start_period: 30s
```

## Expected result

`docker-compose.yml` exists at the repo root.
