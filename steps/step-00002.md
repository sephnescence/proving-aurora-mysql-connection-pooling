# Step 00002 — Create pnpm-workspace.yaml

## Context
Declare the workspace package globs so pnpm resolves all apps and packages as workspace members.

## Action
Create the file `pnpm-workspace.yaml` at the repo root with the following exact content:

```yaml
packages:
  - "apps/*"
  - "packages/*"
```

## Expected result
`pnpm-workspace.yaml` exists at the repo root. Running `pnpm install` from the root will recognise packages under `apps/` and `packages/`.
