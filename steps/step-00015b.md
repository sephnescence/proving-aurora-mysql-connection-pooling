# Step ✅ 00015b — Replace Jest with Vitest in pooled-app

## Context

The NestJS CLI scaffolds a Jest-based test setup by default. This project standardises on Vitest. Remove Jest, ESLint, and related tooling, then add Vitest.

## Action

Replace `apps/pooled-app/package.json` with:

```json
{
  "name": "@pooling-poc/pooled-app",
  "version": "0.0.1",
  "description": "",
  "author": "",
  "private": true,
  "license": "UNLICENSED",
  "scripts": {
    "build": "nest build",
    "start": "nest start",
    "start:dev": "nest start --watch",
    "start:debug": "nest start --debug --watch",
    "start:prod": "node dist/main",
    "test": "vitest run"
  },
  "dependencies": {
    "@nestjs/common": "^11.0.1",
    "@nestjs/core": "^11.0.1",
    "@nestjs/platform-express": "^11.0.1",
    "@nestjs/terminus": "^11.1.1",
    "@pooling-poc/db-query": "workspace:*",
    "nestjs-pino": "^4.6.1",
    "pino": "^10.3.1",
    "pino-http": "^11.0.0",
    "reflect-metadata": "^0.2.2",
    "rxjs": "^7.8.1"
  },
  "devDependencies": {
    "@nestjs/cli": "^11.0.0",
    "@nestjs/schematics": "^11.0.0",
    "@nestjs/testing": "^11.0.1",
    "@types/express": "^5.0.0",
    "@types/node": "^22.19.15",
    "typescript": "^5.7.3",
    "vitest": "^4.1.0"
  }
}
```

Create `apps/pooled-app/vitest.config.ts`:

```typescript
import { defineConfig } from 'vitest/config'

export default defineConfig({
  test: {
    globals: true,
    environment: 'node',
  },
})
```

Delete the generated Jest and ESLint files:

```bash
rm -rf apps/pooled-app/test
rm apps/pooled-app/eslint.config.mjs
rm apps/pooled-app/.prettierrc
```

The root `.prettierrc` applies to the whole repo — the per-app copy is redundant.

Run `pnpm install` from the repo root to update the lockfile.

## Expected result

`apps/pooled-app` has no Jest or ESLint dependencies. `vitest.config.ts` exists. `pnpm --filter @pooling-poc/pooled-app test` runs via Vitest.
