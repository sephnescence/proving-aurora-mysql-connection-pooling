# Step ✅ 00015 — Install @pooling-poc/pooled-app dependencies

## Context

Add the workspace db-query package and additional runtime dependencies needed by the app.

## Commands

Run from the repo root:

```bash
pnpm --filter @pooling-poc/pooled-app add @pooling-poc/db-query@workspace:* @nestjs/terminus nestjs-pino pino pino-http
pnpm --filter @pooling-poc/pooled-app add -D @types/node
```

## Expected result

`apps/pooled-app/package.json` has `@pooling-poc/db-query` listed as a dependency with `"workspace:*"`. The packages are symlinked via pnpm workspaces.
