# Step 00037 — Create Artillery sequential.yml

## Context

3 virtual users, each making 5 sequential requests to `/processlist`. Target URL is injected at runtime via the `TARGET_URL` environment variable. The `X-Test-Type` header is set to `sequential`.

## Action

Create the file `packages/load-tests/sequential.yml` with the following exact content:

```yaml
config:
  target: '{{ $env.TARGET_URL }}'
  phases:
    - duration: 1
      arrivalCount: 3
  defaults:
    headers:
      X-Test-Type: 'sequential'

scenarios:
  - name: 'Sequential processlist'
    flow:
      - loop:
          - get:
              url: '/processlist'
        count: 5
```

## Expected result

`packages/load-tests/sequential.yml` exists.
