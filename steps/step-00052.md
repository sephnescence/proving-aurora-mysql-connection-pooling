# Step 00052 â€” Create .github/workflows/deploy.yml

## Context

Deploy workflow. Triggered manually on `main`. Builds and pushes Docker images to ECR, deploys the CloudFormation main stack, updates kubeconfig, installs Helm charts (ESO, ALB Controller, Fluent Bit), and applies K8s manifests with `envsubst`.

## Action

Create the file `.github/workflows/deploy.yml` with the following exact content:

```yaml
name: Deploy

on:
  workflow_dispatch:
    branches:
      - main

permissions:
  id-token: write
  contents: read

jobs:
  deploy:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - name: Configure AWS credentials
        uses: aws-actions/configure-aws-credentials@v4
        with:
          role-to-assume: arn:aws:iam::${{ secrets.AWS_ACCOUNT_ID }}:role/aurora-pooling-poc-deploy-role
          aws-region: ap-southeast-2

      - name: Login to ECR
        id: login-ecr
        uses: aws-actions/amazon-ecr-login@v2

      - name: Set image tag
        run: echo "IMAGE_TAG=${{ github.sha }}" >> $GITHUB_ENV

      - name: Get ECR URIs from bootstrap stack
        run: |
          ECR_REPO_POOLED=$(aws cloudformation describe-stacks \
            --stack-name aurora-pooling-poc-bootstrap \
            --query "Stacks[0].Outputs[?OutputKey=='ECRPooledAppUri'].OutputValue" \
            --output text)
          ECR_REPO_UNPOOLED=$(aws cloudformation describe-stacks \
            --stack-name aurora-pooling-poc-bootstrap \
            --query "Stacks[0].Outputs[?OutputKey=='ECRUnpooledAppUri'].OutputValue" \
            --output text)
          echo "ECR_REPO_POOLED=$ECR_REPO_POOLED" >> $GITHUB_ENV
          echo "ECR_REPO_UNPOOLED=$ECR_REPO_UNPOOLED" >> $GITHUB_ENV

      - name: Build and push pooled-app image
        run: |
          docker build -f apps/pooled-app/Dockerfile -t $ECR_REPO_POOLED:$IMAGE_TAG .
          docker push $ECR_REPO_POOLED:$IMAGE_TAG

      - name: Build and push unpooled-app image
        run: |
          docker build -f apps/unpooled-app/Dockerfile -t $ECR_REPO_UNPOOLED:$IMAGE_TAG .
          docker push $ECR_REPO_UNPOOLED:$IMAGE_TAG

      - name: Deploy CloudFormation main stack
        run: |
          aws cloudformation deploy \
            --template-file infra/main.yml \
            --stack-name aurora-pooling-poc \
            --capabilities CAPABILITY_NAMED_IAM \
            --parameter-overrides \
              ImageTagPooled=$IMAGE_TAG \
              ImageTagUnpooled=$IMAGE_TAG \
            --region ap-southeast-2

      - name: Update kubeconfig
        run: |
          aws eks update-kubeconfig \
            --name aurora-pooling-poc-eks \
            --region ap-southeast-2

      - name: Get CloudFormation stack outputs
        run: |
          STACK=aurora-pooling-poc
          get_output() {
            aws cloudformation describe-stacks --stack-name $STACK \
              --query "Stacks[0].Outputs[?OutputKey=='$1'].OutputValue" --output text
          }
          echo "POOLED_DB_HOST=$(get_output PooledDbHost)" >> $GITHUB_ENV
          echo "UNPOOLED_DB_HOST=$(get_output UnpooledDbHost)" >> $GITHUB_ENV
          echo "POOLED_DB_SECRET_ARN=$(get_output PooledDbSecretArn)" >> $GITHUB_ENV
          echo "UNPOOLED_DB_SECRET_ARN=$(get_output UnpooledDbSecretArn)" >> $GITHUB_ENV
          echo "ESO_ROLE_ARN=$(get_output EsoRoleArn)" >> $GITHUB_ENV
          echo "ALB_CONTROLLER_ROLE_ARN=$(get_output AlbControllerRoleArn)" >> $GITHUB_ENV
          echo "FLUENT_BIT_ROLE_ARN=$(get_output FluentBitRoleArn)" >> $GITHUB_ENV
          echo "POOLED_LOG_GROUP=$(get_output PooledLogGroupName)" >> $GITHUB_ENV
          echo "UNPOOLED_LOG_GROUP=$(get_output UnpooledLogGroupName)" >> $GITHUB_ENV

      - name: Helm â€” install External Secrets Operator
        run: |
          helm repo add external-secrets https://charts.external-secrets.io
          helm repo update
          helm upgrade --install external-secrets external-secrets/external-secrets \
            --namespace external-secrets-system \
            --create-namespace \
            --wait

      - name: Helm â€” install AWS Load Balancer Controller
        run: |
          helm repo add eks https://aws.github.io/eks-charts
          helm repo update
          helm upgrade --install aws-load-balancer-controller eks/aws-load-balancer-controller \
            --namespace kube-system \
            --set clusterName=aurora-pooling-poc-eks \
            --set serviceAccount.annotations."eks\.amazonaws\.com/role-arn"=$ALB_CONTROLLER_ROLE_ARN \
            --wait

      - name: Helm â€” install Fluent Bit
        run: |
          helm repo add aws https://aws.github.io/eks-charts
          helm repo update
          helm upgrade --install aws-for-fluent-bit aws/aws-for-fluent-bit \
            --namespace amazon-cloudwatch \
            --create-namespace \
            --set serviceAccount.annotations."eks\.amazonaws\.com/role-arn"=$FLUENT_BIT_ROLE_ARN \
            --set cloudWatch.enabled=true \
            --set cloudWatch.region=ap-southeast-2 \
            --set cloudWatch.logGroupName=/aurora-pooling-poc/$(POD_LABEL_APP) \
            --wait

      - name: Apply Kubernetes manifests
        run: |
          for manifest in \
            k8s/namespace.yaml \
            k8s/pooled-app-external-secret.yaml \
            k8s/unpooled-app-external-secret.yaml \
            k8s/pooled-app.yaml \
            k8s/unpooled-app.yaml \
            k8s/ingress.yaml; do
            envsubst < $manifest | kubectl apply -f -
          done

      - name: Wait for deployments to be ready
        run: |
          kubectl rollout status deployment/pooled-app -n pooling-poc --timeout=300s
          kubectl rollout status deployment/unpooled-app -n pooling-poc --timeout=300s
```

## Expected result

`.github/workflows/deploy.yml` exists.
