# Step 00016b — Add root build script and build db-query

## Context

`apps/pooled-app` and `apps/unpooled-app` import from `@pooling-poc/db-query`, which means `db-query` must be compiled to `dist/` before any app can be type-checked or built. Without this, TypeScript cannot resolve the types and the build fails.

`pnpm -r build` runs the `build` script across all workspace packages in topological order — packages are built after their workspace dependencies, so `db-query` is always built before the apps that depend on it.

## Action

Add a `build` script to the root `package.json`:

```json
"scripts": {
  "build": "pnpm -r build",
  "format": "prettier --write ."
}
```

Then build `db-query` for the first time:

```bash
pnpm --filter @pooling-poc/db-query build
```

## Expected result

`packages/db-query/dist/` exists. Running `pnpm build` from the repo root builds all packages in dependency order.
