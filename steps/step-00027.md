# Step 00027 — Create Dockerfile for unpooled-app

## Action
Create the file `apps/unpooled-app/Dockerfile` with the following exact content:

```dockerfile
FROM node:22-alpine AS builder

RUN corepack enable && corepack prepare pnpm@latest --activate

WORKDIR /app

COPY package.json pnpm-workspace.yaml pnpm-lock.yaml ./
COPY tsconfig.base.json ./
COPY packages/db-query/package.json ./packages/db-query/
COPY apps/unpooled-app/package.json ./apps/unpooled-app/

RUN pnpm install --frozen-lockfile

COPY packages/db-query ./packages/db-query
COPY apps/unpooled-app ./apps/unpooled-app

RUN pnpm --filter @pooling-poc/db-query build
RUN pnpm --filter @pooling-poc/unpooled-app build

# ---- runtime ----
FROM node:22-alpine AS runtime

RUN apk add --no-cache wget

RUN corepack enable && corepack prepare pnpm@latest --activate

WORKDIR /app

COPY package.json pnpm-workspace.yaml pnpm-lock.yaml ./
COPY tsconfig.base.json ./
COPY packages/db-query/package.json ./packages/db-query/
COPY apps/unpooled-app/package.json ./apps/unpooled-app/

RUN pnpm install --frozen-lockfile --prod

COPY --from=builder /app/packages/db-query/dist ./packages/db-query/dist
COPY --from=builder /app/apps/unpooled-app/dist ./apps/unpooled-app/dist

EXPOSE 3000

CMD ["node", "apps/unpooled-app/dist/main.js"]
```

## Expected result
`apps/unpooled-app/Dockerfile` exists.
