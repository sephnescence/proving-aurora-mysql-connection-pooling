# Step 00005 — Scaffold @pooling-poc/db-query package

## Context
Create the `packages/db-query` directory and initialise its `package.json`.

## Commands

```bash
mkdir -p packages/db-query/src
cd packages/db-query
pnpm init
```

Then edit `packages/db-query/package.json` so it contains exactly:

```json
{
  "name": "@pooling-poc/db-query",
  "version": "1.0.0",
  "private": true,
  "main": "dist/index.js",
  "types": "dist/index.d.ts",
  "scripts": {
    "build": "tsc",
    "test": "vitest run"
  }
}
```

## Expected result
`packages/db-query/package.json` exists with the name `@pooling-poc/db-query`.
