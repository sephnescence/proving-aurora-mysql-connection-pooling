# ✅ Step 00001 — Initialise pnpm workspace root

## Context

Create the monorepo root `package.json` using pnpm's init command, then configure it as a private workspace root.

## Commands

```bash
pnpm init
```

Then open `package.json` and make the following edits:

- Set `"name"` to `"aurora-pooling-poc"`
- Set `"private"` to `true`
- Remove the `"main"` field (not needed for a workspace root)
- Remove the `"scripts"` block (will be added in later steps as needed)

## Expected result

`package.json` at the repo root with content similar to:

```json
{
  "name": "aurora-pooling-poc",
  "version": "1.0.0",
  "private": true,
  "description": "",
  "keywords": [],
  "author": "",
  "license": "ISC"
}
```
