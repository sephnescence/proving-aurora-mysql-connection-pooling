# Step 00036 — Create Artillery concurrent.yml

## Context

15 virtual users each making 1 simultaneous request to `/processlist`. The `X-Test-Type` header is set to `concurrent`.

## Action

Create the file `packages/load-tests/concurrent.yml` with the following exact content:

```yaml
config:
  target: '{{ $env.TARGET_URL }}'
  phases:
    - duration: 1
      arrivalCount: 15
  defaults:
    headers:
      X-Test-Type: 'concurrent'

scenarios:
  - name: 'Concurrent processlist'
    flow:
      - get:
          url: '/processlist'
```

## Expected result

`packages/load-tests/concurrent.yml` exists.
