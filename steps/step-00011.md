# ✅ Step 00011 — Create DbQueryModule

## Context

The NestJS module that provides and exports `DbQueryService`.

## Action

Create the file `packages/db-query/src/db-query.module.ts` with the following exact content:

```typescript
import { Module } from '@nestjs/common'
import { LoggerModule } from 'nestjs-pino'
import { DbQueryService } from './db-query.service'

@Module({
  imports: [LoggerModule.forRoot()],
  providers: [DbQueryService],
  exports: [DbQueryService],
})
export class DbQueryModule {}
```

## Expected result

`packages/db-query/src/db-query.module.ts` exists.
