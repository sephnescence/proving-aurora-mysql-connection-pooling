# Step 00014 — Scaffold @pooling-poc/pooled-app with NestJS CLI

## Context

Create the `apps/pooled-app` NestJS application using the NestJS CLI. The CLI generates the standard module/controller/service skeleton and `main.ts`.

## Pre-requisite

The NestJS CLI must be available. Install it globally if not already present:

```bash
pnpm add -g @nestjs/cli
```

## Commands

Run from the repo root:

```bash
nest new apps/pooled-app --package-manager pnpm --skip-git --strict
```

When prompted for a package manager, select `pnpm`.

Then update `apps/pooled-app/package.json`:

- Change `"name"` to `"@pooling-poc/pooled-app"`
- Add `"private": true`

Delete the generated placeholder files that will be replaced in later steps:

```bash
rm apps/pooled-app/src/app.controller.spec.ts
rm apps/pooled-app/src/app.controller.ts
rm apps/pooled-app/src/app.service.ts
```

## Expected result

`apps/pooled-app/` exists with a valid NestJS project skeleton. `apps/pooled-app/package.json` has `"name": "@pooling-poc/pooled-app"`.
