# Step ✅ 00046 — Create infra/bootstrap.yml (CloudFormation)

## Context

Bootstrap stack deployed once manually. Creates: deploy IAM role (trusted by GitHub Actions on `main`) and two ECR repositories (pooled-app and unpooled-app). Stack name: `aurora-pooling-poc-bootstrap`.

Note: the GitHub OIDC provider is not created here — it already exists in the account and is referenced directly by ARN in the deploy role trust policy.

## Action

Create the file `infra/bootstrap.yml` with the following exact content:

```yaml
AWSTemplateFormatVersion: '2010-09-09'
Description: Bootstrap stack — deploy role, ECR repos

Parameters:
  GitHubOrg:
    Type: String
    Description: GitHub organisation or username
  GitHubRepo:
    Type: String
    Description: GitHub repository name
    Default: proving-aurora-mysql-connection-pooling

Resources:
  DeployRole:
    Type: AWS::IAM::Role
    Properties:
      RoleName: aurora-pooling-poc-deploy-role
      AssumeRolePolicyDocument:
        Version: '2012-10-17'
        Statement:
          - Effect: Allow
            Principal:
              Federated: !Sub 'arn:aws:iam::${AWS::AccountId}:oidc-provider/token.actions.githubusercontent.com'
            Action: sts:AssumeRoleWithWebIdentity
            Condition:
              StringEquals:
                token.actions.githubusercontent.com:aud: sts.amazonaws.com
              StringLike:
                token.actions.githubusercontent.com:sub: !Sub 'repo:${GitHubOrg}/${GitHubRepo}:ref:refs/heads/main'
      ManagedPolicyArns:
        - arn:aws:iam::aws:policy/AdministratorAccess

  ECRPooledApp:
    Type: AWS::ECR::Repository
    Properties:
      RepositoryName: aurora-pooling-poc/pooled-app

  ECRUnpooledApp:
    Type: AWS::ECR::Repository
    Properties:
      RepositoryName: aurora-pooling-poc/unpooled-app

Outputs:
  DeployRoleArn:
    Value: !GetAtt DeployRole.Arn
    Export:
      Name: aurora-pooling-poc-bootstrap-DeployRoleArn
  ECRPooledAppUri:
    Value: !GetAtt ECRPooledApp.RepositoryUri
    Export:
      Name: aurora-pooling-poc-bootstrap-ECRPooledAppUri
  ECRUnpooledAppUri:
    Value: !GetAtt ECRUnpooledApp.RepositoryUri
    Export:
      Name: aurora-pooling-poc-bootstrap-ECRUnpooledAppUri
  ECRPooledAppName:
    Value: !Ref ECRPooledApp
    Export:
      Name: aurora-pooling-poc-bootstrap-ECRPooledAppName
  ECRUnpooledAppName:
    Value: !Ref ECRUnpooledApp
    Export:
      Name: aurora-pooling-poc-bootstrap-ECRUnpooledAppName
```

## Expected result

`infra/bootstrap.yml` exists.
