# ✅Step 00007 — Create packages/db-query/tsconfig.json

## Context

TypeScript config for the db-query package, extending the root base config. NestJS requires `experimentalDecorators` and `emitDecoratorMetadata`.

## Action

Create the file `packages/db-query/tsconfig.json` with the following exact content:

```json
{
  "extends": "../../tsconfig.base.json",
  "compilerOptions": {
    "experimentalDecorators": true,
    "emitDecoratorMetadata": true,
    "outDir": "dist",
    "rootDir": "src"
  },
  "include": ["src"],
  "exclude": ["node_modules", "dist"]
}
```

## Expected result

`packages/db-query/tsconfig.json` exists and extends the root base config.
