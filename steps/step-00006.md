# ✅ Step 00006 — Install @pooling-poc/db-query dependencies

## Context
Install runtime and dev dependencies for the db-query package.

## Commands

Run from the repo root:

```bash
pnpm --filter @pooling-poc/db-query add @nestjs/common @nestjs/core mysql2 nestjs-pino pino pino-http reflect-metadata rxjs
pnpm --filter @pooling-poc/db-query add -D typescript @types/node vitest @nestjs/testing
```

## Expected result
`packages/db-query/package.json` now has `dependencies` and `devDependencies` sections. A `pnpm-lock.yaml` is created (or updated) at the repo root.
