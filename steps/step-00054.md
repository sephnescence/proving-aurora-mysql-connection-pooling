# Step 00054 — Create .github/workflows/teardown-bootstrap.yml

## Context
Bootstrap teardown workflow. Triggered manually on `main`. Guards against the main stack still existing, empties ECR repositories (required before deletion), then deletes the bootstrap stack.

## Action
Create the file `.github/workflows/teardown-bootstrap.yml` with the following exact content:

```yaml
name: Teardown Bootstrap

on:
  workflow_dispatch:
    branches:
      - main

permissions:
  id-token: write
  contents: read

jobs:
  teardown-bootstrap:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - name: Configure AWS credentials
        uses: aws-actions/configure-aws-credentials@v4
        with:
          role-to-assume: arn:aws:iam::${{ secrets.AWS_ACCOUNT_ID }}:role/aurora-pooling-poc-deploy-role
          aws-region: ap-southeast-2

      - name: Guard — main stack must not exist
        run: |
          STATUS=$(aws cloudformation describe-stacks \
            --stack-name aurora-pooling-poc \
            --query "Stacks[0].StackStatus" \
            --output text 2>/dev/null || echo "DOES_NOT_EXIST")
          if [ "$STATUS" != "DOES_NOT_EXIST" ]; then
            echo "ERROR: aurora-pooling-poc stack still exists (status: $STATUS). Run teardown.yml first."
            exit 1
          fi
          echo "Main stack does not exist. Proceeding."

      - name: Get ECR repo names from bootstrap stack
        run: |
          ECR_POOLED=$(aws cloudformation describe-stacks \
            --stack-name aurora-pooling-poc-bootstrap \
            --query "Stacks[0].Outputs[?OutputKey=='ECRPooledAppName'].OutputValue" \
            --output text)
          ECR_UNPOOLED=$(aws cloudformation describe-stacks \
            --stack-name aurora-pooling-poc-bootstrap \
            --query "Stacks[0].Outputs[?OutputKey=='ECRUnpooledAppName'].OutputValue" \
            --output text)
          echo "ECR_POOLED=$ECR_POOLED" >> $GITHUB_ENV
          echo "ECR_UNPOOLED=$ECR_UNPOOLED" >> $GITHUB_ENV

      - name: Empty pooled-app ECR repository
        run: |
          IMAGE_IDS=$(aws ecr list-images \
            --repository-name $ECR_POOLED \
            --query "imageIds[*]" \
            --output json)
          if [ "$IMAGE_IDS" != "[]" ]; then
            aws ecr batch-delete-image \
              --repository-name $ECR_POOLED \
              --image-ids "$IMAGE_IDS"
          fi

      - name: Empty unpooled-app ECR repository
        run: |
          IMAGE_IDS=$(aws ecr list-images \
            --repository-name $ECR_UNPOOLED \
            --query "imageIds[*]" \
            --output json)
          if [ "$IMAGE_IDS" != "[]" ]; then
            aws ecr batch-delete-image \
              --repository-name $ECR_UNPOOLED \
              --image-ids "$IMAGE_IDS"
          fi

      - name: Delete bootstrap CloudFormation stack
        run: |
          aws cloudformation delete-stack \
            --stack-name aurora-pooling-poc-bootstrap \
            --region ap-southeast-2

      - name: Wait for bootstrap stack deletion
        run: |
          aws cloudformation wait stack-delete-complete \
            --stack-name aurora-pooling-poc-bootstrap \
            --region ap-southeast-2
```

## Expected result
`.github/workflows/teardown-bootstrap.yml` exists.
