# ✅ Step 00003 — Create .gitignore

## Context
Prevent generated artefacts and secrets from being committed.

## Action
Create the file `.gitignore` at the repo root with the following exact content:

```
# dependencies
node_modules/

# build output
dist/

# environment
.env

# pnpm
.pnpm-store/

# editors
.DS_Store
```

## Expected result
`.gitignore` exists at the repo root. The `.env` file and all `node_modules/` directories are ignored by git.
