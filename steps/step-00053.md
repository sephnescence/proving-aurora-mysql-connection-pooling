# Step 00053 — Create .github/workflows/teardown.yml

## Context

Teardown workflow. Triggered manually on `main`. Deletes Kubernetes resources first (to release ALB ENIs), waits for ENIs to be released, then deletes the CloudFormation main stack.

## Action

Create the file `.github/workflows/teardown.yml` with the following exact content:

```yaml
name: Teardown

on:
  workflow_dispatch:
    branches:
      - main

permissions:
  id-token: write
  contents: read

jobs:
  teardown:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - name: Configure AWS credentials
        uses: aws-actions/configure-aws-credentials@v4
        with:
          role-to-assume: arn:aws:iam::${{ secrets.AWS_ACCOUNT_ID }}:role/aurora-pooling-poc-deploy-role
          aws-region: ap-southeast-2

      - name: Update kubeconfig
        run: |
          aws eks update-kubeconfig \
            --name aurora-pooling-poc-eks \
            --region ap-southeast-2

      - name: Delete Kubernetes resources
        run: |
          kubectl delete ingress,deployment,service --all -n pooling-poc --ignore-not-found=true

      - name: Wait for ALB ENIs to be released
        run: |
          VPC_ID=$(aws cloudformation describe-stacks \
            --stack-name aurora-pooling-poc \
            --query "Stacks[0].Outputs[?OutputKey=='VpcAId'].OutputValue" \
            --output text 2>/dev/null || echo "")

          if [ -z "$VPC_ID" ]; then
            echo "Could not determine VPC ID, waiting 60s as fallback"
            sleep 60
          else
            echo "Polling for ENI release in VPC $VPC_ID"
            for i in $(seq 1 20); do
              ENI_COUNT=$(aws ec2 describe-network-interfaces \
                --filters "Name=vpc-id,Values=$VPC_ID" "Name=description,Values=ELB*" \
                --query "length(NetworkInterfaces)" \
                --output text)
              echo "Iteration $i: $ENI_COUNT ELB ENIs remaining"
              if [ "$ENI_COUNT" -eq 0 ]; then
                echo "All ENIs released"
                break
              fi
              sleep 15
            done
          fi

      - name: Delete CloudFormation main stack
        run: |
          aws cloudformation delete-stack \
            --stack-name aurora-pooling-poc \
            --region ap-southeast-2

      - name: Wait for stack deletion
        run: |
          aws cloudformation wait stack-delete-complete \
            --stack-name aurora-pooling-poc \
            --region ap-southeast-2
```

## Expected result

`.github/workflows/teardown.yml` exists.
