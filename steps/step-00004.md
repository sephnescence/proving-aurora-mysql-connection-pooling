# ✅ Step 00004 — Create root tsconfig.base.json

## Context
Shared TypeScript compiler options inherited by all workspace packages.

## Action
Create the file `tsconfig.base.json` at the repo root with the following exact content:

```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "CommonJS",
    "moduleResolution": "node",
    "lib": ["ES2022"],
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
    "resolveJsonModule": true,
    "declaration": true,
    "declarationMap": true,
    "sourceMap": true
  }
}
```

## Expected result
`tsconfig.base.json` exists at the repo root and will be extended by each package's own `tsconfig.json`.
