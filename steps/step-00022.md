# Step 00022 — Configure nestjs-pino in unpooled-app main.ts

## Action

Replace the entire contents of `apps/unpooled-app/src/main.ts` with:

```typescript
import { NestFactory } from '@nestjs/core'
import { Logger } from 'nestjs-pino'
import { AppModule } from './app.module'

async function bootstrap() {
  const app = await NestFactory.create(AppModule, { bufferLogs: true })
  app.useLogger(app.get(Logger))
  await app.listen(3000)
}

bootstrap()
```

## Expected result

`apps/unpooled-app/src/main.ts` uses `nestjs-pino` Logger and listens on port 3000.
