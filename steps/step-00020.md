# Step 00020 — Create Dockerfile for pooled-app

## Context
Multi-stage Docker build for the pooled NestJS app. Base image is `node:22-alpine`. `wget` must be present for the Docker Compose health check.

## Action
Create the file `apps/pooled-app/Dockerfile` with the following exact content:

```dockerfile
FROM node:22-alpine AS builder

# Install pnpm
RUN corepack enable && corepack prepare pnpm@latest --activate

WORKDIR /app

# Copy workspace manifests
COPY package.json pnpm-workspace.yaml pnpm-lock.yaml ./
COPY tsconfig.base.json ./
COPY packages/db-query/package.json ./packages/db-query/
COPY apps/pooled-app/package.json ./apps/pooled-app/

# Install dependencies
RUN pnpm install --frozen-lockfile

# Copy source
COPY packages/db-query ./packages/db-query
COPY apps/pooled-app ./apps/pooled-app

# Build db-query first, then pooled-app
RUN pnpm --filter @pooling-poc/db-query build
RUN pnpm --filter @pooling-poc/pooled-app build

# ---- runtime ----
FROM node:22-alpine AS runtime

RUN apk add --no-cache wget

RUN corepack enable && corepack prepare pnpm@latest --activate

WORKDIR /app

COPY package.json pnpm-workspace.yaml pnpm-lock.yaml ./
COPY tsconfig.base.json ./
COPY packages/db-query/package.json ./packages/db-query/
COPY apps/pooled-app/package.json ./apps/pooled-app/

RUN pnpm install --frozen-lockfile --prod

COPY --from=builder /app/packages/db-query/dist ./packages/db-query/dist
COPY --from=builder /app/apps/pooled-app/dist ./apps/pooled-app/dist

EXPOSE 3000

CMD ["node", "apps/pooled-app/dist/main.js"]
```

## Expected result
`apps/pooled-app/Dockerfile` exists.
